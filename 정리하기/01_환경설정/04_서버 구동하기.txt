/ 04_서버 구동하기 /

이제 장고로 로컬서버가 구축되었다. 서버에 접속을 해보자

  1. 서버에 접속하기 
  2. 서버 종료하기


/ -- 1. 서버에 접속하기 / 


-- python manage.py runserver

Ex)
  (mysite) C:\projects\mysite>python manage.py runserver

python 명령어 / manage.py 프로젝트로 만들어진 매니저 파일로 / runserver 서버를 실행한다. 

서버를 실행하면 http://로 이루어진 주소가 뜨는데, 그 주소를 복사해서 인터넷에 치면 장고로 만들어진 서버가 실행된다.

http://127.0.0.1:8000/


/ -- 2. 서버 종료하기 /

명령프롬프트에서 실행중인 서버에서 

Ctrl + C 

를 2번 입력하면 서버를 종료할 수 있다.
