/ 05_가상환경 PATH만들기 /

가상환경에 진입하기 위해서는 가상환경에 진입하기 위해서 가상환경 폴더안에 직접적으로 다 들어가야하는 귀찮음이 존재한다. 그걸 배제하기 위해서 빠르게 진입할 수 있는 PATH를 만들어 보자

/ cmd 파일 만들기 / 

mysite.txt 우선 텍스트 파일을 먼저 만들자 

텍스트 파일안에 명령어를 작성하자

Ex) 
    @echo off
    cd cd:/projects/mysite
    c:/venvs/mysite/scripts/activate

장고로 만들어진 프로젝트폴더에 진입하고, 가상환경 프로그램을 실행한다는 명령어이다.

이렇게 작성하고 확장자를 cmd파일로 변경하자

만들어진 mysite.cmd파일을 가상환경 mysite폴더에 넣자.


/ 환경변수 설정하기 /

window + R을 눌러서 

sysdm.cpl을 실행하자


1. window + R 을 눌러서 시스템 실행창을 뛰운다.
2. sysdm.cpl 을 적고 실행한다.
3. 시스템 속성이라는 창이 뜨면 고급탭에 들어간다.
4. 고급탭에서 환경변수를 누른다.
5. PC에 대한 사용자 변수와 시스템 변수에 PATH를 찾는다.
6. PATH를 더블클릭하면 여러가지 PATH들이 나온다.
7. PATH에서 새로만들기를 눌러서 아까 만들어 두었던 mysite.cmd파일의 경로를 입력하자
